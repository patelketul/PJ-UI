A. Project Title: Selenium automation testing
B. Getting started:
1. Install any IDE supporting Java. For example Intellij IDEA
2. Add all jar files from Extlib folder inside project folder to projects class path

C. Run test cases:
1. Test cases are written in "src > Tests > MainTest.java" file. Run MainTest class to run test cases
2. Provide account username and password in "testdate.json" file under "config" folder.
